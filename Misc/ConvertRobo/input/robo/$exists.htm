$exists

Purpose

Check if the node exists in the parse tree.

Syntax

returnedBoolean = variableType("$exists"[, number])

returnedBoolean - type: bool
variableType - type: N
number - type: int

Returns

Returns 1 if the node exists and 0 if the node does not exist.
